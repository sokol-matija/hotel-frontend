# CLAUDE.md

> AI assistant instructions for create. Keep this file updated as the project evolves.

## Stack

- **Vite 6 + React 19 + TypeScript** (strict mode, all checks enabled)
- **Tailwind CSS v4** (CSS-first config in `src/index.css`, no `tailwind.config.js`)







## Code conventions

- All imports use `@/` alias (never `../../` beyond 1 level up)
- No default exports for utilities/hooks — named exports only
- Route files use default export (TanStack Router convention)
- Double quotes everywhere (Prettier enforces)
- No `any`, no `// @ts-ignore`, no `console.log` in committed code
- `src/components/feature-name/` for feature components
- `src/components/ui/` for ShadCN primitives only (never modified manually)
- Co-locate tests: `Button.tsx` + `Button.test.tsx`

## Testing

- Unit tests: Vitest + Testing Library (test behavior, not implementation)
- E2E tests: Playwright with Page Object Model (`tests/pages/`)
- Never mock Supabase in unit tests — use fixtures
- Coverage thresholds: 70% lines/functions/branches

## DO NOT

- Start the dev server (hot reload is already running)
- Modify `src/routeTree.gen.ts` — it's auto-generated by TanStack Router
- Commit `.env` files
- Use `!important` in CSS
- Add dependencies without checking bundle impact (`npm run knip`)

## Commands

```bash
npm run dev              # start dev server (already running)
npm run validate:script  # full quality pipeline
npm run test:e2e:ui      # Playwright UI
npm run release:dry      # preview next semver release

```
